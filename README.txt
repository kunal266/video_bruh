Video Summarizer

> Note: This branch is for linux & mac users. For windows users checkout the `windows` branch, though I'm not too sure of that atm.

-------------

To install:
    - pip install -r requirements.txt
    - brew install ffmpeg (for mac users) OR sudo apt install ffmpeg (for ubuntu users)

To run:
    - Execute `app.py` present in the root directory via `python3 app.py`
